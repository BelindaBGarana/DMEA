How to Use: Web App

The easiest way to use DMEA is with the web application here: https://belindabgarana.shinyapps.io/dmea/

To use the web app, format your input drug rank list or gene signature as a .txt file which looks like the examples below. DMEA is flexible to accept any signed numeric values as rank metrics for drugs or genes. For example, drug ranks could be outputs from CMap queries and gene ranks could be Log2FC values from analysis of a GEO dataset. For the drug rank list input, the third column with mechanism-of-action (moa) annotations is optional. You can also test DMEA using one of the provided examples.

Please access this website on desktop to view these examples.

For more information, please watch our demo video below or Contact Us.

How to Use: R Package

DMEA is intended to be user-friendly. To use DMEA, please install our R package by running the 2 lines of code below and then use the following instructions to either input a drug rank list or a gene signature. For more information, visit our GitHub repo here: https://github.com/BelindaBGarana/DMEA

if (!require(devtools)) {install.packages(dev.tools) }

devtools::install_github ('BelindaBGarana/DMEA')

Drug Rank List

  1. Install the DMEA R package and load into your library
  2. Input a drug rank list directly into the drugSEA function, as well as a gmt object of drug sets (such as our provided gmt file)
  3. Save the output dataframe of the enrichment results for each drug set in addition to plots summarizing the results

Gene Signature

  1. Install the DMEA R package and load into your library
  2. Input a gene signature directly into the DMEA function as well as a gmt object of drug sets (such as our provided gmt file) and dataframes of drug sensitivity and gene expression (e.g., PRISM and CCLE)
  3. Save the output dataframes with the Weighted Voting, correlation, and enrichment results for each drug set in addition to plots summarizing the results
